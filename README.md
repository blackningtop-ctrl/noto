# Noto — 무료 Notion 스타일 워크스페이스

벤토 그리드 UI를 갖춘 **완전 무료 · 로컬 저장** Notion 클론입니다.  
로그인, 서버, 구독 없이 브라우저만으로 모든 기능을 사용할 수 있습니다.

## 기능

| 영역 | 내용 |
|------|------|
| **홈** | 벤토 그리드 대시보드 (최근 문서, 즐겨찾기, 빠른 생성) |
| **페이지** | 중첩 페이지, 아이콘, 커버, 즐겨찾기, 복제 |
| **블록 에디터** | 텍스트, H1–H3, 목록, 할 일, 인용, 코드, 콜아웃, 구분선, 이미지, 토글 |
| **슬래시 명령** | `/` 로 블록 타입 선택 |
| **마크다운 단축** | `# ` `## ` `- ` `[] ` `> ` ` ``` ` `---` |
| **데이터베이스** | 테이블 · 칸반 보드 · 갤러리, 속성 추가 |
| **검색** | `Ctrl/⌘ + K` 전역 검색 |
| **휴지통** | 삭제 · 복원 · 영구 삭제 |
| **테마** | 라이트 / 다크 |
| **백업** | JSON 내보내기 / 가져오기 |

데이터가 **localStorage**에만 저장되므로 완전 비공개이며 비용이 들지 않습니다.

## 시작하기

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 을 엽니다.

```bash
npm run build    # 프로덕션 빌드
npm run preview  # 빌드 미리보기
```

## 기술 스택

- React 19 + TypeScript + Vite
- Tailwind CSS 4
- Zustand (persist)
- Lucide icons
